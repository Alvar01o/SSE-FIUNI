# SSE-FIUNI
Sistema de seguimiento de egresados FIUNI


## Instrucciones
```sh
git clone https://github.com/Alvar01o/SSE-FIUNI.git
cd SSE-FIUNI
docker-compose build
docker-compose up
```
Entrar en el servidor 

```sh
sh build.sh
```
